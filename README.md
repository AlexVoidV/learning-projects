# Collection of personal mini-projects in Python

**To install:** use `git clone` and `uv sync` (or use `pip install`)

You can run files via `pip` by activating your **.venv** first, or use `uv run` without activation.

## Calculator GUI
Launch by `uv run main.py`

## To-Do CLI
Launch by `uv run main.py <command>`, where the `command` is:
- `help`, `add <task>`, `ls`, `delete <task number>` or `delete 0` (to delete all tasks).

## To-Do GUI
Launch by `uv run main.py`

## Password Generator
Launch by `uv run main.py`
