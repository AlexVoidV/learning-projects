# Collection of personal mini-projects in Python

**To install:** use `git clone` and `uv sync`

## Calculator GUI
Launch by `uv run main.py`

## To-Do CLI
Launch by `uv run main.py <command>`, where the `command` is:
- `help`, `add <task>`, `ls`, `delete <task number>` or `delete 0` (to delete all tasks).
